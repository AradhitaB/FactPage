# FactPage

FactPage is a very benign FactPage. Look through Facts until you are bored or we run out of new Facts. Then, click on a button to get more Facts... or rather stats about users browsing fact page.

Each user is presented with a webpage that has one of four possible design combinations. Interactions with the page are recorded and used to do a nifty A/B test that users are welcome to peruse. 

Seeing empty stats? Don't worry, we just haven't gotten enough users yet. FactPage will grow as the users do.

# How it works

Mix together a SQLite database, a uvicorn supported backend, some analytics and a streamlit. Et voila - FactPage.
